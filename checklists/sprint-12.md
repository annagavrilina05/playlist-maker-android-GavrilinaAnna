Чек-лист для самопроверки

Экран «Избранное»
+ Есть отдельный экран Favorites.
+ На экране отображается список всех избранных треков.
+ Каждый элемент списка показывает обложку, название, исполнителя и время.
+ Если у трека нет обложки, отображается заглушка.
+ Долгое нажатие на трек удаляет его из «Избранного».
+ Наверху экрана есть иконка «Назад» для возврата.

Дополнительная фича: Вариант 1. Удаление плейлиста:
+ При нажатии на иконку корзины на экране плейлиста плейлист удаляется.
+ При долгом нажатии на плейлист в списке появляется возможность удалить его.
+ После удаления пользователь возвращается на экран списка плейлистов.
+ Удалённый плейлист отсутствует и в списке, и в базе данных.
