# playlist-maker-android-GavrilinaAnna
Учебный проект по созданию мобильного приложения

## О проекте

Playlist Maker - это мобильное приложение для поиска музыки, создания и управления персональными плейлистами.

## Текущий прогресс

### Спринт 3
- Создана базовая структура проекта
- Реализован главный экран приложения
- Добавлены экраны Search и Settings
- Настроена навигация между экранами

### Спринт 4
- Реализована навигация между экранами с использованием Jetpack Navigation
- Создан PlaylistHost с NavHost для управления переходами
- Доработан экран поиска:
    - Добавлена поисковая строка с иконками лупы и очистки
    - Реализована история поисковых запросов
    - Настроен интерфейс с заголовком и кнопкой назад
- Улучшен экран настроек:
    - Добавлен переключатель темы
    - Реализован функционал шеринга приложения
    - Настроена отправка email в поддержку
    - Добавлено открытие пользовательского соглашения
- Обновлены ресурсы приложения
- Улучшена архитектура с использованием единой MainActivity
